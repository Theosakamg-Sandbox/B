1. Création du projet

Dans un dossier ExamGit, créer un projet ayans la structure suivante : 

|- src/
|   |- main/
|   |    |- java/
|   |    |- html/
|   |    |- ressources/
|   |    |     |- css/
|   |    |     |- js/
|   |- test/
|   |- vendor/
|- doc/
|   |- UML/
|   |- User_manual/
|- README.md
|- LICENSE.md
|- AUTHORS.md
|- CONTRIBUTING.md
|- CODE_OF_CONDUCT.md

Dans le fichier README.md : 

Exam GIT 2019 - Mon Site Web.

Dans le fichier LICENSE.md : copier/coller le contenue de la license Apache V2, comme dans le TP1.

Dans le fichier AUTHORS.md : Ajouter votre Nom Prenom.

Dans le fichier CONTRIBUTING.md : 

Any contribution that you make to this repository will

[license](http://www.apache.org/licenses/LICENSE-2.0.html):

2. Depot GIT

Initialisez le dépot de versionning
Conservez un historique avec pour message "Initial commit".
Créer un depot distant sur Github et déployer.

3. Ignorer les fichiers du depot

Exclure les fichiers de type :

  • jar
  • war
  • iml
Exclure les fichiers pour le language JAVA.
Exclure les fichiers émanent d'un MacOS(DS_Store...) & Windows(Thumbs.db...)

Conservez un historique avec pour message "Ignore untracked files".
Vous pouvez utiliser les sites github.com ou gitignore.io

4. Ajouter une etiquette de Version

Ajouter un etiquette de version 0.0.1 et deployé sur le depot distant.

5. Ajouter des projets externes

Ajouter dans le dossier vendor les projets suivant :

  • https://github.com/twbs/bootstrap & Conservez un historique avec pour message "Add vendor sub-modules bootstrap".
  • https://github.com/jquery/jquery & Conservez un historique avec pour message "Add vendor sub-modules jquery".
  • https://github.com/Theosakamg-Sandbox/B Conservez un historique avec pour message "Add vendor sub-modules Theosaka".

6. Créer votre page portfolio.

Dans une branche "CV", créer un page HTML dans src/main/html/cv.html avec votre CV comportant :

  • Identité (Nom, prenom, age.)
  • Vos qualifications (Diplome préparer ou obtenue)
  • Vos expériences (projet réaliser par ordre chronologiaue)
Conservez un historique avec pour message "Add Portfolio". Puis fusionner dans la branche Master.

7. Créer une page du Centre de formation.

Dans une branche "Institut", créer un page HTML dans src/main/html/institut.html comportant :

  • Identité (Nom, address)
  • Liste des formations
  • Ajouter un lien du centre de formation de votre page cv.html ver la page institut,
Conservez un historique avec pour message "Add Institut". Ne pas fusionner la branche pour le moment.

8. Ajouter une etiquette de Version

Ajouter un etiquette de version 0.1.0 et deployé sur le depot distant.

9. Ajouter votre page portfolio a votre voisin (de gauche).

apres qu'il vous soit donné acces a sont depot...
Dans une branche "voisin_left", ajouter dans src/main/html/voisin.html une copie de votre page cv.html . Conservez un historique avec pour message "Add Voisin page".

10. Ajouter un page par defaut.

Dans une branche "index", créer un page HTML dans src/main/html/index.html avec :

  • Un slogan et une image
  • Listes des liens vers les autres pages
Conservez un historique avec pour message "Add Index page". Puis fusionner dans la branche Master.

11. Intéger les pages en attente.

Fusionner les branche :

  • Voisin
  • Institut
