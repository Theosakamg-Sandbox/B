Si aucune branch n'est evoqué utiliser la master !!!

1. Ajouter Bootstrap.

Dans une branche "add_vendor", ajouter bootstrap (lien vers vendor/bootstrap/dist/css/*.css) a vos pages :

  • cv , Conservez un historique avec pour message "Add Design for CV"
  • institut, Conservez un historique avec pour message "Add Design for Institut"
  • index , Ne pas conserver pour le moment dans l'historique.
  • voisin_* , Ne pas conserver pour le moment dans l'historique
Suivre le tuto du chapitre "Bootstrap 4 Basic Template"

2. Ajouter vos voisins comme auteurs.

Dans une branche "Update_author" basé sur la master sans vos modification de design(bootstrap),
Ajouter dans le fichier AUTHORS.md, le nom de vos voisins ayant ajoutés leurs pages dans votre dépot.
Conservez un historique avec pour message "Add new authors".

3. Personaliser le design.

Dans la branche "add_vendor", Changer la couleur de fond du bloc "collapsibleNavbar" en vert (#34ab1d) dans un fichier CSS (dans src/main/ressource/css/main.css)
Conservez un historique avec pour message "Add final Design".

4. Rebaser la branche.

Rebaser la branche "add_vendor" sur la base de la branch "master". ATTENTION !!!

4. Ajouter Jquery.

Dans une branche "add_vendor", ajouter Jquery : jquery-3.4.1.min.js dans le dossier src/main/ressources/js/
Conservez un historique avec pour message "Building Jquery".
Ajouter la librarie dans toutes les pages de votre site. Activer un slide-show
